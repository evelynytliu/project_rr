# 轉圈圈滑步車餐館 (Round & Round Cafe) - 品牌全案設計紀錄

這是一個關於「轉圈圈滑步車餐館」的品牌全案設計作品集網站。網站內容涵蓋了該品牌的創業歷程、空間設計、視覺識別、行銷活動及周邊商品設計。

## 專案內容

本專案完整記錄了從零開始打造一個親子友善餐廳的過程，包括：

*   **品牌故事 (Brand Story)**: 結合滑步車運動與餐飲的理念。
*   **空間設計 (Space Design)**: 店面外觀、室內動線與色彩規劃。
*   **視覺識別 (Visual Identity)**: Logo、菜單設計、插畫與黑板畫。
*   **行銷活動 (Marketing)**: 票券設計、宣傳單與社群經營。
*   **周邊商品 (Merchandise)**: 紅包袋、毛巾、獎盃等實體設計。

## 技術架構

*   **HTML5 / CSS3**: 語意化標籤與響應式設計。
*   **Tailwind CSS**: 快速構建現代化、RWD 的使用者介面。
*   **JavaScript**: 處理捲動動畫 (Reveal on Scroll) 與互動效果。
*   **Google Fonts**: 使用 Noto Sans TC 與 Noto Serif TC 呈現質感排版。

## 如何瀏覽

直接開啟 `index.html` 即可瀏覽完整網頁內容。
